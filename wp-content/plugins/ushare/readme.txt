追求最佳用户体验的社交分享插件，为你的博客提供大多数国内社交平台内容分享功能。

== Description ==

ushare is a social share plugin with flat design, it supports most social networks.(优享是针对中国国内的一款社交分享插件，采用简约的扁平化设计风格，并支持常用的社交分享网络。)

== Installation ==

1. 上传到 /wp-content/plugins/ 目录
2. 在后台插件菜单激活该插件
3. 在后台 设置>优享配置 页面设置插件选项
